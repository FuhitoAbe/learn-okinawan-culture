#Overall
ポップコーンメーカーを使用した、沖縄の文化に関するクイズ(問題)を行う。

事前に教師が問題を記述した動画を作成し、学生は問題の答えを検索を利用して探し出す。

音や動きを動画で見ることで、より身近に沖縄の文化を感じてもらえることを目的とする。また、同時に回答を模索する過程で情報検索能力も養う。
